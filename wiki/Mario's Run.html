Mario's Run

Synopsis: Drop blocks and springs onto the course to influence the outcome of the game as Mario walks back and forth like a Goomba.

Release Date: June 6, 2007

Genre: Minigame

Download on itch.io

Note: This game is included in Hello Winter Mario Games 2009, which is included in the Hello Fangaming Time Capsule.

Trivia

  • The sprites used when Mario gets squished in Mario's Run were taken from Super Mario War. Unfortunately, this means squishing Mario in this game is not an original idea.
  • Just like Hello Fangaming's previous minigame, this game only has one level.
  • This game was later included in Hello Winter Mario Games 2009, with a wintery theme, playable Luigi, and other improvements.
