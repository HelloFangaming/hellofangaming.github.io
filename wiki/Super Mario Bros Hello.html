Genre: Platform

Download on Indie DB

Note: This game is included in the Hello Fangaming Time Capsule.

Trivia

  • This is Hello Fangaming's second game to use Trisoft's engine.
  • Includes various code changes and improvements to the engine.
  • New features were added to Trisoft's engine for this game.
  • Includes 3 worlds with 4 levels each, for a total of 12 levels.
