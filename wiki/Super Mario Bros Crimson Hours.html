Super Mario Bros: Crimson Hours

Synopsis: Mario with guns and blood! And explosions!

Release Date: September 21, 2007

Genre: Shooter

Download on itch.io

Note: This game is included in the Hello Fangaming Time Capsule.

Trivia

  • The game starts off with Bowser killing Princess Peach. Mario then goes off on a revenge journey.
  • Guns and blood make a comeback, but this time, with a health system, difficulty options, and grenades!
  • This game is Hello Fangaming's second shooter game. The next one to be made is Super Mario Bros: Bloody Battles.
  • This game was created using an open source SMB2 engine Hello Fangaming released, since the main engine was in rough shape at the time.
