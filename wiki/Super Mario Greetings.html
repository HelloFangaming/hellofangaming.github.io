Super Mario Greetings

Synopsis: Mario needs to manipulate switches in order to collect all the red coins.

Release Date: September 1, 2006

Genre: Platform/Puzzle

Download on itch.io

Note: This game is included in the Hello Fangaming Time Capsule.

Trivia

  • This is Hello Fangaming's last game to use Trisoft's engine.
  • This is Hello Fangaming's first game to feature a puzzle environment with collectable objects that are required to beat each level. This concept was explored again later in Super Mario Bros: Coin Quest.
  • This is Hello Fangaming's first game to feature a world map, use Black Squirrel's tileset, and include remixed music.
  • Super Mario Greetings was built right on top of Super Mario Bros: Hello's source, which was built using Mario's Journey's source.
