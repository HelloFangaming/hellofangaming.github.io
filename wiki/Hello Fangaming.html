Hello Fangaming (ヘロー・ファンゲーミング) is a weeaboo game developer that is focused on making anime games. Games for weebs by weebs!

History

Hello Fangaming is a long-time fangame developer that has been developing fangames since 2006, and started off with the creation of Mario fangames using GameMaker. More detailed information on these older games can be found in the Hello Fangaming Collection, which is not only a collection of old remastered levels, but a game that serves as an autobiography on past Hello Fangaming games. Eventually, Hello Fangaming has made the transition from GameMaker to Unity, and from Mario fangames to anime games, which is the current focus.
