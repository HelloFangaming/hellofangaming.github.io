Hello Fangaming Time Capsule + Launcher

Synopsis: A decade's worth of fangames, all in one download! The Hello Fangaming Time Capsule + Launcher includes a grand total of 40 fangames, all of which can be accessed from one convenient launcher.

Release Date: August 15, 2017

Genre: Time Capsule

Download on Indie DB

Trivia

  • This time capsule includes all of the games that made an appearance in the Hello Fangaming Collection.
  • The Hello Fangaming Time Capsule + Launcher made an appearance in the Nintendo Community Fangame Convention in 2017. You can still view its booth here.
  • The launcher was coded in C# using Microsoft Visual Studio.
  • Overwhelmed by the number of games included in this time capsule? Check out the Time Capsule Game Recommendations page to get an idea of what game to play.
