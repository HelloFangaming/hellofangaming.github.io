Super Mario Bros: Adventure Journey

Synopsis: Mario goes on an adventure and a journey.

Release Date: December 14, 2006

Genre: Platform

Download on itch.io

Note: This game is included in the Hello Fangaming Time Capsule.

Trivia

  • This is the first of six games Hello Fangaming created using the Ultramario engine.
  • The Ultramario engine was still fairly new at the time, so numerous features added to the Ultramario engine later on were not included.
  • This is the last fangame released by Hello Fangaming in 2006.
  • The first level of this game was later recreated in Hello Fangaming Collection as Snow Summit.
