Shroom Editor

Synopsis: The Shroom Project meets Mario Editor! The resistance is back, and this time, you get to create their adventures! Build your own levels with guns and blood with this easy to use level editor!

Want to share your level with others? Want to try out levels others have made? Check out the Shroom Editor Showcase!

Release Date: February 14, 2018

Genre: Shooter/Level Editor

Trivia

  • Shroom Editor combines the level editor from Mario Editor with the shooter gameplay from The Shroom Project.
  • Just like Mario Editor, this game also includes a comprehensive tutorial level in order to teach new players how to use its various features. In addition to this, a short level called The Resistance Returns is included in order to reintroduce the characters.
  • This game contains eight playable characters, more than any other Hello Fangaming game. Six recurring NPC characters from The Shroom Project are now playable for the first time.
