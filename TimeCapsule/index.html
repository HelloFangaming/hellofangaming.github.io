About

It's time to explore Hello Fangaming's kurorekishi.

Experience all of Hello Fangaming's old games with infinite lives, less slippery player physics, pixel perfect fullscreen scaling, and more with the Hello Fangaming Time Capsule. Isekai Mario is not included in this time capsule due to its file size being larger than the entire time capsule. Make sure to download it separately for the complete Hello Fangaming Mario experience. Three of the games require the Microsoft Visual C++ 2010 SP1 Redistributable Package (x86) to be installed. Make sure to do that. Most games do not have gamepad support. You can download AntiMicroX and use it to map your gamepad's buttons to keyboard keys. Please note that the majority of the games included in this time capsule were make by a kid with too much free time. Try not to take them too seriously!

Hello Fangaming 20th Anniversary Time Capsule Released

March 7, 2025

The Hello Fangaming Time Capsule returns to celebrate Hello Fangaming's 20th Anniversary. 34 games have been recompiled with various improvements, making this the definitive way to experience Hello Fangaming's old games!

  • New and improved launcher with a dark theme.
  • Added Super Mario Prism and The Shroom Project.
  • Redundant games have been removed. Super Crown Land and Toadette's Sweets Gallery are not included due to their inclusion in Isekai Mario. Mario's Run and Super Luigi Paradise have been removed due to their inclusion in Hello Winter Mario Games 2009.
  • All legacy GameMaker (6.1-8.0) games have been recompiled with GameMaker 8.2, an unofficial update to GameMaker, which make the games run better on modern PCs.
  • The recompiled games have been bundled with dgVoodoo which makes them use modern versions of DirectX.
  • All known memory leaks have been patched.
  • The following games have been updated to be less slippery by backporting the values for acceleration from newer games: SMB: Restless Reality, Hello Winter Mario Games 2009, SMB: Seeking Sunshine, SMB: Bloody Battles, SMB: Shine Expedition, Super Luigi and the Golden Shrooms, SMB: Fierce Days, SMB: Revenge of Bowser, SMB: Dual Dash, SMB: Crimson Hours, SMB: Shine Pursuit, SMB: Kingdom Troubles, and SMB: Koopa Chaos.
  • Some games were updated to allow you to save your progress more often rather than only at castles. The save system from SMB: Bowser's Terror was backported to all Ultramario Engine games.
  • All games can now be played with infinite lives!
  • All games released before 2011 now use the 2011 Hello Fangaming logo. (The newer 2023 logo is reserved for anime games!)
  • All save files are now neatly stored in AppData. No more highscores being saved to the Windows registry.
  • No more forced fullscreen, resolution swapping, frame skipping, 4:3 aspect ratio games being stretched wide, or other funny business.
  • SMB: Ztar Turmoil no longer displays your PC's account name during the intro. (Better for streamers!)
  • GM: Studio Scheduler fix by omicronrex is now bundled with all GM: Studio games.
